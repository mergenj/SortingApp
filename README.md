# SortingApp
This application lets the user to input an array of integers and pick an algorithm to sort them.

# How to use?
There are 2 versions of application available.
#
If you are using the version with GUI run from AppGUI.java , input an array of integers separated with ,[comma] for example: 34,12,41 then pick sorting algorithm from dropdown box and click "Sort" button.

For the console version run from App.java, read the instructions at run time.
#
#
